JavaScript Cheat Sheet
  • Falsy values: A value is considered "falsy" if it translates to false when evaluated in a boolean context. The following values are considered falsy in JavaScript:
    • false
    • 0 (zero)
    • -0 (negative zero)
    • 0n (BigInt zero)
    • "" or '' or `` (empty string)
    • null
    • undefined
    • NaN (Not-A-Number)
  • Truthy values: Any value that is not falsy is considered "truthy." This means that almost every other value will be evaluated as true in a boolean context. Some common examples of truthy values include:
    • true
    • Any number other than 0
    • Any non-empty string
    • Infinity and -Infinity
    • Most values of Symbol and BigInt
    • Objects are always true unless it's explicitly null or undefined
  • Variables (let, const, var): JavaScript has three ways to declare variables:
    • let (block-scoped, mutable)
    • const (block-scoped, immutable)
    • var (function-scoped, now mostly discouraged due to issues with hoisting).
  • Data types:
    • There are 7 primitive types: string, number, bigint, boolean, symbol, null and undefined.
    • And one non-primitive data type: object for more complex data structures.
  • Operators:
    • typeof, instanceof, this, new, delete, if, else
    • ? (conditional operator), ?. (optional chaining), ?.(), ?.[]
    • == (equality regardless of type -> type conversion), === (equality and types match)
    • ! (NOT), ?? (Nullish Coalescing), && (AND), || (OR)
    • try, catch, finally, async, await, class, interface

  • Objects
    • Stores properties as key value pairs in {...}.
    • Objects are passed by reference, whereas primitives are passed by value.
    • Can be created with {}. Functions are also objects.
    • Object declaration, initialization, usage

                    id: "1",
                    name: "Andrew",
                    role: "Admin",
                  }

                  keys = Object.keys(user); // [id, name, role]

  • Arrays
    • Array Declaration: 
                      
                  let arr = new Array();
                  let arr = [];
    • Array Initialization: 
                      
                  let arr = ["Apple", "Banana", "Pear"];
                  // Arrays can store elements of any type
                  let arr = ["Apple", { name: "Andrew" }, true, function() { console.log("Hello World") } ];
    • Array Operations: 
                      
                  let apple = arr[0];
                  let banana = arr[1];
                  let pear = arr[2];
                  arr[2] = "Plum"; // replace an element
                  arr[3] = "Lemon"; // add a new one to the array
                  for (let i = 0; i < arr.length; i) {
                    console.log(arr[i]);
                  }
                  for (let fruit of arr) {
                    console.log(fruit);
                  }
                  let undef = arr[-1]; // not allowed in JavaScript, the result will be undefined
  • Functions:
    • Function Declaration: 
                      
                  function greet(name) {
                    return `Hello, ${name}!`;
                  }
    • Function Expression: 
                      
                  // JavaScript functions are values too
                  const greet = function(name) {
                    return `Hello, ${name}!`;
                  }
    • Arrow Function: 
                      
                  const greet = (name) => {
                    return `Hello, ${name}!`;
                  }
    • Shortened Arrow Function: 
                      
                  const greet = name => return `Hello, ${name}!`;
    • Callback function: passing function as parameter 
                      
                  // asynchronous callback
                  function fetchData(callback) {
                    setTimeout(() => {
                      const data = "Here is your data!";
                      callback(data);
                    }, 2000);
                  }     
                  function displayData(data) {
                    console.log(data);
                  }
                  fetchData(displayData); 
                  
                  // event handling callback
                  const button = document.getElementById('myButton');
                  button.addEventListener('click', function() {
                    // function definition
                  });

Built-In Functions
Method Description
alert() Displays a message in a modal dialog box. Use sparingly, as it interrupts user experience.
confirm() Displays a dialog box with a message and "OK" and "Cancel" buttons. Returns true if the user clicks "OK" and false otherwise.
eval() Executes a string of JavaScript code. Avoid using eval() due to potential security and performance risks.
isFinite() Checks if a value is a finite number. Converts the argument to a number if necessary.
isNaN() Checks if a value is NaN (Not-a-Number).
parseFloat() Converts a string to a floating-point number.
parseInt() Converts a string to an integer. Can specify a radix (base) for parsing.
prompt() Displays a dialog box that asks for user input. Returns the input as a string. Often disabled or discouraged for modern web development.
decodeURI() Decodes a Uniform Resource Identifier (URI) encoded with encodeURI().
decodeURIComponent() Decodes a URI component encoded with encodeURIComponent().
encodeURI() Encodes a URI by escaping characters that are not valid in a URI.
encodeURIComponent() Encodes a URI component by escaping characters that have special meaning.
setTimeout() Executes a function after a specified delay (in milliseconds).
clearTimeout() Cancels a setTimeout timer.
setInterval() Executes a function repeatedly, with a fixed time delay between calls.
clearInterval() Cancels a setInterval timer.
