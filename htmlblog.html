HTML Cheat Sheet

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. "Hypertext" refers to links that connect web pages to one another, either within a single website or between websites.

HTML Syntax

  • <!DOCTYPE html> at the top level declares the document type and version of HTML
  • Space and indentation in code is ignored in HTML. It only affects readability.
  • HTML elements are the building blocks of HTML. Each element begins with a opening tag <tag> and ends with a closing tag <tag/>.
  • Void elements have self-enclosing tags (no closing tag) and no content. <img src="image.jpg" alt="Description" >
  • Head section is for code that will not be displayed on browser.
  • Body section is for code that will be rendered as content on browser.
  • HTML enables elements to be nested within other elements. 
                  
                1 <div>
                2  <h1>Title</h1>
                3  <p>This is a paragraph inside a div.</p>
                4 </div>
  • Comments are written like this: <!--This is a comment -->
  • JavaScript scripts imported is preferably at the end of the body so that HTML is rendered first, otherwise loading scripts would delay HTML content and the web page may be disoriented.
